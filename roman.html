Convert English Numbers to Roman Numerals

A JavaScript application to convert English numbers to Roman Numerals

  • Take a user input & convert to integer value
  • Pass the value through a function
  • Return the value as Roman Numerals
  • NB: Big numbers have a dot after them:
  • .4k = MV. |.. 5k = V. | 10k = X. | 50k = L.
  • 100k = C. | 500k = D. | 1m = M.
  • NB: Long numbers will bleed of the edge!

Type a positive number in the box, or alternatively use the scrolly arrows.
